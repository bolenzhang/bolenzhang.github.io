go语言特点

  • 没有对象，没有继承多态，没有泛型，没有try/catch
  • 有接口，函数式编程，CSP并发模型（goroutine + channel）

课程概览

  1. 基本语法：变量、选择、循环、指针、数组、容器
  2. 面向接口： 结构体、duck typing的概念、组合的思想
  3. 函数式编程：闭包的概念、多样的立体
  4. 工程化：资源管理、错误处理、测试和文档、性能调优
  5. 并发编程：goroutine和channel、理解调度器、多样的例题。

课程里基本没用锁的概念

  1. 实战项目：从0开始，使用go语言自主搭建简单分布式爬虫，爬取相亲网站的资料。

一、基本语法：变量

1. 变量

1.1 注意

  • 变量可以放在包内定义，也可以放在函数内定义。
  • 函数外面定义变量必须加var关键字，形如a:=3只能在函数内用. 而且这是包内部变量，go语言没有全局变量的说法。
  • 可以用var()集中定义变量

1.2 内建变量类型

  • bool, string

  • (u)int, (u)int8, (u)int16, (u)int32, (u)int64, uintptr

    • int类型分为两类，1类是规定长度的int8\int16…，另一类是不规定长度的int，它是由操作系统位数自动决定的，go中没有long类型的。
    • go语言中也有指针uintptr, 比c语言中的好用。

  • byte,rune

    • rune 是go语言中字符类型，4字节，go语言中不叫char了，char只有一字节造成很多坑，所以就用rune。utf-8编码中很多都是3字节的。因此rune采用32位的int32。
    • 他们跟整数都是可以混用的，相当于整数的别名。

  • float32， float64, complex64, complex128

    • complex复数，比如complex64实部和虚部分别是32位，go是目前唯一把复数作为内建类型的语言，为了方便工程中的建模等等。
    • 负数回顾：
      • i = 根号1
      • 复数：3+4i.
    • 注意float类型表示数字时经常不准（在所有语言中都是这样），所以很多时候要用类似% .xf表示。如下欧拉公式如果不取前三位就打印不出来0.

##二、条件语句

2.1 if

  • if的条件里可以赋值
  • if的条件里赋值的变量作用域就在这个if语句里

2.2 switch

  • switch会自动break, 除非使用fallthrough
  • switch后可以没有表达式

三、循环语句

for 可以省略初始条件、递增条件、结束条件。go中用for 后不跟任何条件来作死循环，没有while循环。

小结

  • for, if 后面的条件没有括号
  • if 条件里也可以定义变量
  • 没有while
  • switch不需要break， 也可以总结switch多个条件
